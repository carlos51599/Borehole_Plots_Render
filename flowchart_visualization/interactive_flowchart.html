🚀 Interactive Codebase Explorer - User Guide

🎯 Navigation

  • Click & Drag: Pan around the diagram
  • Mouse Wheel: Zoom in/out
  • Click Node: Select and highlight dependencies
  • Double-click Node: Focus on specific file
  • Hover Node: View detailed information

🔍 Features

  • Search: Find files or functions by name
  • Filter: Show specific file types
  • Layout: Change visualization style
  • Dependencies: Red lines show import relationships
  • Complexity: Node size indicates code complexity

🎨 Visual Elements

  • Node Colors: Represent different file types
  • Node Size: Proportional to code complexity
  • Edge Thickness: Indicates dependency strength
  • Highlighting: Shows related files when selected

⚡ Keyboard Shortcuts

  • Space: Reset view
  • F: Focus mode
  • H: Toggle help
  • T: Toggle theme
  • Escape: Clear selection
